
A simple Rendering framework for openGL
- Includes Phong lighting
- Specular light source is moveable using left mouse button (drag) and scroll (for up and down)
- Includes textures included in mtl file
- Tested on Obj models
- Camera controls- W,S,A,D,Q,E

Dependencies:
- Uses Assimp 3.1 for loading models. Library is statically linked
- Uses SOIL for loading textures
- Uses glew32 and glfw
